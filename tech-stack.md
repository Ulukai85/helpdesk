# Tech Stack

## Frontend

- **Framework:** React with TypeScript
- **Routing:** React Router
- **Styling:** Tailwind CSS

## Backend

- **Runtime:** Node.js with TypeScript
- **Framework:** Express

## Database

- **Database:** PostgreSQL
- **ORM:** Prisma

## Authentication

- Database sessions

## AI

- **Provider:** Anthropic Claude API
- Classification, summarization, and suggested replies

## Email

- **Provider:** SendGrid or Mailgun
- Inbound webhook for receiving support emails
- Outbound API for sending replies

## Deployment

- Docker
